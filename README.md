## 🔍 Key Insights

- **Total Customers:** 116K
- **Churned Customers:** 14K (12.19% Churn Rate)
- **Retained Customers:** 102K (87.81% Retention Rate)
- **Total Loan Loss:** $35.88M
- **Top Churn Reasons:**  
  - Service Issues (25.35%)  
  - Account Closure (25.17%)  
  - Relocation (25.03%)  
  - Better Offers Elsewhere (24.44%)
